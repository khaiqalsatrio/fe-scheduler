# Planner

Folder ini berisi dokumentasi dan rencana terkait fitur pengelolaan jadwal (Planner) di aplikasi ChatAja!.

## Fitur
1. **Get All Agendas**: Mengambil semua jadwal kegiatan dari server.
2. **Create Agenda**: Menambahkan kegiatan baru ke server melalui modal Tambah Aktivitas.
3. **Day Selector**: Memilih hari kegiatan.
3. **Timeline View**: Menampilkan kegiatan berdasarkan waktu.
