# Todo

Folder ini berisi dokumentasi terkait tugas-tugas (Todo) yang perlu dikerjakan oleh pengguna.

## Fitur Utama
- [x] List Chat (Conversation)
- [x] Detail Chat & History Chat
- [x] Kirim Pesan Teks
- [x] Kirim Pesan Media (Gambar, File, Suara)
- [x] Reset Chat (Hapus & Buat Ulang) [/]
- [ ] Group Management

## Integrasi
Fitur Todo akan dihubungkan dengan agenda harian. Setiap agenda baru yang dibuat dapat memiliki daftar tugas spesifik yang harus diselesaikan.
